## Comprehensive List of Functional Programming (FP) Topics

**I. Foundations & Paradigm**

1.  **FP Principles:** immutability, first-class & higher-order functions, pure functions, referential transparency.
2.  **Mathematical Background:** lambda calculus, category theory (functors, monads), combinatory logic.
3.  **Evaluation Strategies:** eager vs lazy, normal order vs applicative order.

**II. Core Concepts**

4.  **Functions as Values:** composition, currying, partial application.
5.  **Recursive Thinking:** structural & tail recursion, recursion schemes (fold, unfold, map, reduce).
6.  **Algebraic Data Types (ADTs):** product types (records/tuples), sum types (variants), pattern matching.
7.  **Type Systems:** Hindleyâ€“Milner, type inference, parametric polymorphism.

**III. Abstractions & Structures**

8.  **Functors, Applicatives, Monads, Arrows, Monoids, Semigroups.
9.  **Monoid Homomorphisms, Monoids in parallelism (foldMap).**
10. **Lenses, optics, immutable data structures.

**IV. Effect Management**

11. **IO encapsulation, monad transformers, algebraic effects.**
12. **Purely functional data structures (persistent, finger trees).**

**V. Languages & Ecosystem**

13. **Pure FP Languages:** Haskell, Elm, PureScript, Idris.
14. **Multi-paradigm FP:** Scala, F#, OCaml, Elixir, Clojure, Kotlin, Rust.
15. **JavaScript FP libraries:** Ramda, Folktale, RxJS.

**VI. Concurrency & Parallelism**

16. **STM, actors (Akka), functional reactive programming (FRP).**
17. **Lazy data streams, pipelines.

**VII. Testing & Proving**

18. **Property-Based Testing (QuickCheck, Hypothesis).**
19. **Formal verification, dependent types, Agda, Coq.

**VIII. Resources:** "Structure & Interpretation of Computer Programs", "Category Theory for Programmers", "Learn You a Haskell", "Functional Programming in Scala".
